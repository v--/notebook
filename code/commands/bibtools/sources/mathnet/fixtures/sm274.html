О неравенствах для производных колмогоровского типа

Г. Г. Магарил-Ильяевa, В. М. Тихомировb

a Московский государственный технический университет радиотехники, электроники и автоматики
b Московский государственный университет им. М. В. Ломоносова
PDF полного текста (464 kB) PDF английской версии (368 kB) Список цитирования (31)

Аннотация: Общая задача о неравенствах для производных состоит в оценке нормы некоторой производной функции через нормы других ее производных. Цель работы – рассмотрение подобных задач с общих позиций теории экстремума.
